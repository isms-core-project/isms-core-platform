## 6. Compliance Scoring Methodology

### 6.1 Item-Level Scoring

Each assessment item scored as:
- **Implemented (100%)**: Fully compliant, evidence provided
- **Partially Implemented (50%)**: Partially compliant, gaps exist
- **Not Implemented (0%)**: Non-compliant or no evidence
- **Not Applicable (N/A)**: Excluded from scoring

### 6.2 Domain-Level Scoring

Domain Compliance % = SUM(Item Scores) / COUNT(Applicable Items)

### 6.3 Overall Assessment Scoring

Overall Compliance % = AVERAGE(Domain Compliance %)

### 6.4 Traffic Light Thresholds

- 🟢 GREEN (≥80%): Compliant
- 🟡 YELLOW (60-79%): Partially Compliant
- 🔴 RED (<60%): Non-Compliant

"""

Usage: python excel_sanity_check_a828.py [workbook.xlsx]

import openpyxl
import sys

def check_workbook(filename):
    """Run all validation checks"""
    wb = openpyxl.load_workbook(filename)
    
    checks = [
        check_required_sheets(wb),
        check_header_formatting(wb),
        check_formulas(wb),
        check_dropdowns(wb),
        check_sheet_links(wb)
    ]
    
    passed = sum(checks)
    total = len(checks)
    
    print(f"\n{'='*60}")
    print(f"Sanity Check Results: {passed}/{total} checks passed")
    print(f"{'='*60}")
    
    return passed == total

def check_required_sheets(wb):
    """Verify required sheets exist"""
    required = ["Instructions", "Summary Dashboard", "Evidence Register", "Gap Analysis", "Approval Sign-Off"]
    missing = [s for s in required if s not in wb.sheetnames]
    
    if missing:
        print(f"❌ Missing sheets: {missing}")
        return False
    print("✅ All required sheets present")
    return True
