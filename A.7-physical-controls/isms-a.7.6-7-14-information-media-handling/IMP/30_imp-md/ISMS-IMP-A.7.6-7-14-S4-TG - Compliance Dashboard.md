## Assessment Integration

### Data Flow Diagram

```
Monthly Data Sources
├── Clear Desk Audits → Audit Results → Dashboard (Pass Rate)
├── Endpoint Management → Screen Lock Reports → Dashboard (Compliance %)
├── Incident Management → Secure Area Incidents → Dashboard (Count)
└── Asset Management → Disposal Events → Dashboard (Count, Certificate Rate)

Quarterly Assessments
├── ISMS-IMP-A.7.6-7-14-S1 (Secure Areas)
│   └── Summary Dashboard → Overall A.7.6 Score → Dashboard
├── ISMS-IMP-A.7.6-7-14-S2 (Clear Desk/Screen)
│   └── Summary Dashboard → Overall A.7.7 Score → Dashboard
└── ISMS-IMP-A.7.6-7-14-S3 (Equipment Disposal)
    └── Summary Dashboard → Overall A.7.14 Score → Dashboard

Dashboard Consolidation
└── Overall Compliance Score = (A.7.6 × 0.30) + (A.7.7 × 0.35) + (A.7.14 × 0.35)
```

**Weighting Rationale:**
- A.7.6 (30%): Secure areas - foundational but fewer touchpoints
- A.7.7 (35%): Clear desk/screen - daily operational control
- A.7.14 (35%): Equipment disposal - critical data protection at end of lifecycle
