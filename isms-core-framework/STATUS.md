## 📌 What "Complete" means in this file

- **Control pack complete** means: the pack has IMP and SCR coverage, and any required POL is present for governance.
- **Annex A control mapped** means: each of the 93 controls is mapped to at least one pack in `COVERAGE.md` (Primary/Supporting).

This status file tracks **control packs**, not ISO certification status.
