# Code of Conduct

ISMS CORE is an engineering-first security project. We aim for a professional, respectful environment focused on high-quality, verifiable work.

## Our Standards

We expect everyone to:
- Be respectful and constructive
- Assume good intent and focus on facts
- Keep discussions technical and evidence-based
- Respect privacy and confidentiality
- Avoid harassment, discrimination, or hostile behavior

## Scope

This code of conduct applies to:
- Issues, pull requests, code review discussions
- Documentation feedback
- Any project communication spaces

## Enforcement

Project maintainers may take action when behavior undermines a safe, productive environment. Actions may include warning, content removal, or removal of access.

## Reporting

If you experience or witness unacceptable behavior, contact: **info@isms-core.com**
