## 9. Consolidation Script Specifications

### 9.1 Script Requirements

**Script Name:** `generate_a59_4_dashboard.py`

**Key Functions:**

1. **find_latest_workbook(pattern, directory)**
   - Search for workbooks matching pattern
   - Return most recent by modification time
   - Return None if not found

2. **validate_workbook_schema(workbook_path, schema_key)**
   - Check workbook has expected sheets
   - Return (is_valid, missing_sheets, error_message)

3. **extract_score_from_cell(workbook_path, sheet_name, cell_ref)**
   - Open workbook in data_only mode
   - Extract numeric value from cell
   - Handle percentage values (decimal → percentage)
   - Return None on error

4. **count_assets_in_range(workbook_path, sheet_name, data_range, id_column)**
   - Count non-empty cells in ID column within range
   - Return integer count

5. **count_gaps_by_priority(workbook_path, sheet_name, data_range)**
   - Count gaps where Status ≠ "✅ Closed"
   - Group by Priority (Critical, High, Medium, Low)
   - Return dict: {'critical': X, 'high': Y, 'medium': Z, 'low': W}

6. **consolidate_assessment_data(workbook_paths)**
   - Call all extraction functions
   - Calculate overall score
   - Generate key findings
   - Return consolidated dictionary

7. **create_executive_summary_sheet(wb, consolidated_data)**
   - Generate Sheet 2 with layout specified above

8. **create_*_sheet()** (for each sheet)

9. **main()**
   - Orchestrate entire dashboard generation
   - Handle errors gracefully
   - Provide detailed console output

### 9.2 Error Handling

**Missing Workbooks:**
- Script continues with warnings
- Uses default/placeholder values (0 or N/A)
- Lists missing workbooks in console and Data Sources sheet

**Invalid Workbook Structure:**
- Validates sheet names before extraction
- Reports specific missing sheets
- Skips extraction from invalid workbooks

**Cell Extraction Failures:**
- Logs warning to console
- Returns None
- Overall score calculation handles None values gracefully

**No Valid Data:**
- Dashboard still generates
- Shows "N/A" or 0 values where appropriate
- Instructions explain workbooks need completion

## 10. Quality Assurance

### 10.1 Pre-Generation Validation

**Checklist before running script:**
- [ ] All 3 assessment workbooks completed
- [ ] Quality & Compliance workbook has scores populated
- [ ] All mandatory fields in assessment workbooks filled
- [ ] Workbooks saved with proper filenames
- [ ] All workbooks in same directory as script

### 10.2 Post-Generation Validation

**Dashboard Quality Checks:**
- [ ] Executive Summary displays all scores (no "N/A" except for missing data)
- [ ] Overall Compliance Score calculated correctly (weighted average)
- [ ] Traffic light indicators correct (🟢/🟡/🔴 based on thresholds)
- [ ] Asset counts match source workbooks
- [ ] Gap counts match Gap Analysis sheet
- [ ] Data Sources sheet shows "✅ Loaded" for all workbooks
- [ ] Key Findings auto-generated appropriately
- [ ] No formula errors (#REF!, #NAME?, #VALUE!)
- [ ] All sheets present and named correctly

**Audit Readiness:**
- [ ] Can trace each score back to source cell in assessment workbook
- [ ] Data Sources sheet documents exact files used
- [ ] Timestamp shows when dashboard was generated
- [ ] Findings supported by quantitative data

### 10.3 Common Issues and Solutions

**Issue:** "Overall Compliance: N/A"
- **Cause:** Quality & Compliance workbook not found or scores not populated
- **Solution:** Complete WB3 assessment, ensure file exists

**Issue:** "Total Assets: 0"
- **Cause:** Information/Associated Assets workbooks not found
- **Solution:** Generate WB1 and WB2, populate with asset data

**Issue:** Score extraction returns None
- **Cause:** Cell reference in WORKBOOK_SCHEMAS doesn't match actual location
- **Solution:** Open workbook, verify cell references, update WORKBOOK_SCHEMAS dictionary

**Issue:** Dashboard doesn't update after changing assessment data
- **Cause:** Cached .xlsx files or script not re-run
- **Solution:** Delete old dashboard, re-run script: `python3 generate_a59_4_dashboard.py`

1. **Single Source of Truth:** Dashboard consolidates from 3 assessment workbooks
2. **Automated Extraction:** Python script reads actual workbook data (no manual copying)
3. **Evidence-Based:** Every score traces back to assessment evidence
4. **Executive Focus:** One-page view for decision-makers
5. **Audit Ready:** Data Sources sheet provides full traceability

**Success Criteria:**

✅ Dashboard generates without errors  
✅ All scores extracted correctly from source workbooks  
✅ Traffic light indicators reflect actual compliance status  
✅ Gap summary actionable (priorities, counts, timelines)  
✅ Data Sources sheet shows all workbooks loaded  
✅ CISO can present Executive Summary to Board within 5 minutes  

**Maintenance Cycle:**

- **Weekly:** Address critical/high priority gaps
- **Monthly:** Update asset inventories as changes occur
- **Quarterly:** Full assessment cycle, regenerate dashboard, update Trend Analysis
- **Annually:** Comprehensive review, external audit presentation
